# BunioX Tech Solutions - Corporate Website

A modern, fully responsive website for **BunioX Tech Solutions** - a technology company offering software development, business branding, tech accessories sales, and digital skills training.

## Quick Start

Place the `BunioX-Tech` folder in your `htdocs` directory and access via `http://localhost/BunioX-Tech/`

## Pages

| Page        | Description                                      |
|-------------|--------------------------------------------------|
| `index.html`  | Homepage with hero, services, projects, and shop |
| `about.html`  | Company story, mission, values, and team         |
| `services.html` | Detailed service offerings                     |
| `projects.html` | Filterable project portfolio                   |
| `training.html` | Course listings with pricing (₦)               |
| `shop.html`   | Product store with add-to-cart                   |
| `contact.html`| Contact form and business info                   |

## Tech Stack

- **HTML5** - Semantic markup
- **CSS3** - Custom properties, flexbox, grid, animations
- **Bootstrap 5.3** - Grid system and components
- **Vanilla JavaScript** - Lightweight interactions (no frameworks)
- **Google Fonts** - Poppins font family

## Key Features

- Mobile-first responsive design
- Smooth scroll animations (Intersection Observer)
- Animated statistics counters
- Project category filtering
- Client-side form validation
- Cart notification system
- Sticky shrinking navbar
- Back-to-top button

## Project Structure

```
BunioX-Tech/
├── index.html, about.html, services.html, projects.html,
│   training.html, shop.html, contact.html
├── assets/
│   ├── css/style.css       # Main styles
│   ├── css/pages.css       # Page-specific styles
│   ├── js/main.js          # JavaScript interactions
│   └── images/             # All media files
└── data/                   # JSON data files
```

## Customization

**Colors**: Edit CSS variables in `assets/css/style.css`

**Content**: Update text, prices, and contact details directly in HTML files

**Images**: Replace placeholder elements with `<img>` tags pointing to `assets/images/`
